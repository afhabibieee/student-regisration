# student-regisration

Submission aplikasi sederhana yang <b>mengimplementasikan CRUD dari firebase</b> untuk UAS Mata Kuliah Pemograman Bergerak.

## Features
* Splash screen,
* Onboard Page
* Home Page,
* Tambah Data Page,
* Daftar Siswa Page
* Pop up dialog untuk delete dan update data
